Task 2.2: Time-frequency analysis vs. Waveform

One of the advantages of a time-frequency analysis over a waveform based analysis is that it is an efficient tool for getting more information. More specifically, from the time-frequency plot, we can see the change of the chords through time. Additionally, each note in each chord is displayed clearly in the spectrogram. The information is easy to read - with the black background indicating silence. We know those loud events appear to be bright. In contrast, traditional waveform based analysis only shows how amplitude changes over time in a bipolar manner. The spectrogram is thus more informative and instructive. For instance, from the waveform of Dresscode, we can tell how loud the signal from the perceived amplitude, and we can also see the frequency from wavelength - the longer the wavelength, the lower the frequency. However, we cannot get any detailed information on individual frequency or how it changes over time. When looking at the spectrogram of the same song, we can view a range of frequencies, and by looking at the changes of colors, we know how loud the events are and where the chord changes.
